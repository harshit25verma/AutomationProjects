## Overview

This project focuses on **automated data extraction** from the Polish tech job market
using modern browser automation tools.

It demonstrates **cross-browser testing**, **headless automation**, and **CI integration**.

---

## Automation Approaches

### Selenium (Python)
- Scrapes job titles
- Simulates real user browser behavior
- Useful for legacy or browser-specific testing

### Playwright (Python, Async)
- Scrapes salary data
- Faster execution
- Modern async automation model

---

## CI/CD Integration

A GitLab CI pipeline demonstrates:
- Multi-runtime automation execution
- Headless browser testing
- Automation in containerized CI environments
