# Automation Projects Portfolio

This repository contains multiple automation-focused projects demonstrating
**QA, SDET, system automation, and reliability engineering skills**.

Each project is isolated in its own folder while sharing a common goal:
building reliable, testable, and maintainable automation systems.

---

## Projects

### 1️. [System File Intelligence](https://github.com/harshit25verma/AutomationProjects/tree/9b1029ad7cc4ff45b0989af80bfe635d61371de0/file-intelligence)
A Python-based system maintenance tool that automatically organizes files and validates correctness using Robot Framework.

**Skills demonstrated:**
- OS-level automation
- File system reliability
- Python scripting
- Robot Framework testing
- Monitoring-oriented design

---

### 2️. [Job Market Scraper](https://github.com/harshit25verma/AutomationProjects/tree/9b1029ad7cc4ff45b0989af80bfe635d61371de0/job-market-scraper)

**Skills demonstrated:**
- Web automation
- Cross-browser testing

- Async scraping
- CI/CD with GitLab

## Why a Single Repository?

These projects are grouped together to reflect:
- Shared automation philosophy
- Easier navigation for reviewers
- Clear separation via folder-level boundaries

This mirrors how automation teams often organize internal tooling.
